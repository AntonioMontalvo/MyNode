Using File System Methods

Node's File I/O methods are frequesntly used to read and write files. For a complete list of options go to the docs.

We know that with Node.js we can create a server and use the url to create routes. But normally we wouldn't ask the client to type anything in the url. That is not friendly at all. What we would do is create a page to help the user input data or click here and there to navigate or submit information.

In your terminal type node manyServers.js. Now open a new page in the browser and type http://localhost:8000. The page that you see was read using node's file system method fs.readFile() That is pretty cool.
